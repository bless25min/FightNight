## Default Mode

Default to 0-to-1 Exploration for positioning, landing page structure, creative sections, and event/pass framing.

Default to 1-to-100 Optimization for registration, checkout, payment, tracking, event capacity, confirmation, and production flow.

---

## Core Positioning

FightNight is not a normal fitness class.

It is a pass-based night combat experience.

The page should feel like an entry into a different state, not a gym timetable.

---

## Main Conversion Path

Ad / social traffic
→ event landing page
→ Pass / registration CTA
→ registration / payment
→ confirmation
→ LINE / CRM / event follow-up

---

## Protected Areas

Do not modify without explicit permission:

- registration form
- checkout
- payment
- event capacity
- date / time / location
- Meta Pixel
- LINE / CRM follow-up
- confirmation message
- tracking events

---

## Copy Rules

Prefer:

- Pass
- 入場
- 今晚進入狀態
- 壓力釋放
- 集體能量
- 夜間
- 拳、汗、聲音、節奏

Avoid:

- 健身課
- 燃脂
- 打造更好的自己
- 一般團課
- 高效訓練
- 適合所有人
- generic gym marketing

---

## Design Rules

The design should feel like:

- night event
- fight poster
- premium pass
- controlled intensity
- strong mobile CTA path

Avoid:

- generic fitness templates
- cute wellness style
- long explanation before CTA

---

## Definition of Done

Confirm:

- page does not feel like a generic fitness class
- Pass / event framing remains strong
- event details are easy to find
- CTA path is clear on mobile
- registration / payment / tracking were preserved if present

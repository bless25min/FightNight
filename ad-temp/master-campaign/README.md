# Fight Night Pass 廣告母版

這組是 1080x1080 方形廣告母版，重點不是再讓生成模型硬湊完整廣告，而是把廣告大師應該掌控的幾件事分開：

- 影像：使用最接近真實場館與品牌情緒的專案素材。
- 版面：用 SVG 控制暗部、層次、焦點與 CTA。
- 中文：由設計層排版，不交給生成模型亂寫。
- 測試：四張分別對應不同心理入口，而不是只換文案。

版本：

1. `01-tonight-state.svg`：下班後進入狀態。
2. `02-friend-night.svg`：朋友局，比吃飯更有記憶點。
3. `03-release-pressure.svg`：把今天的悶打出去。
4. `04-first-rhythm.svg`：第一次也能被節奏帶起來。

這些檔案可直接用瀏覽器打開檢視；若要投放，可再輸出成 PNG/JPG。

# FightNight Ad Drafts

這個資料夾用來暫存廣告生圖與後製草稿。

## 保存規則

- 每次使用 image generation 產出的圖片，都要複製到這裡。
- `generated-archive/` 保存原始生成圖，方便回看每一輪失敗原因。
- 後續若有可投放候選圖，另存為 `candidate-*.png`。
- 不直接覆蓋舊圖，用時間或版本命名。

## 目前調整方向

- 不再把女性用戶理解成「女生笑著運動」。
- 廣告要先解除進場焦慮，再建立好玩感。
- 畫面要像首頁的強設計海報，不是單純健身照片。
- AI 不應負責細節可信度最高的部分：手、拳套、臉、真實 logo、格鬥動作。
- 更好的路線是用首頁視覺語法做正方形廣告：黑底敘事區、紅色符號、強標題、真實場館動作圖。

# Fight Night 活動場次／費用資訊頁面稿

## 文件用途
這份文件整理兩個頁面的正式文案：

1. 首頁活動資訊入口
2. `/offers` 頁的 LINE 會員專屬內容頁

這次不再讓首頁直接顯示價格方案，而是改成：

- 首頁只顯示入口 CTA
- 進入頁面後先看到 `LINE 會員專屬內容`
- 使用者完成 LINE Login 後，再解鎖活動場次、費用資訊、教練資訊與名額狀態

## 一、首頁活動資訊入口

### 區塊定位
首頁不再顯示價格與方案卡，而是只保留一個明確入口，讓使用者先進入查看活動資訊。

### 區塊標題
`查看活動場次 / 費用資訊`

### 區塊副標
`首頁不直接顯示價格與方案。進入頁面後，再用 LINE Login 解鎖完整內容。`

### 區塊補充說明
`你會看到 LINE 會員專屬內容入口。快速登入後，可查看活動場次、費用資訊、教練資訊與可報名名額。`

### 區塊中段標題
`這個頁面會解鎖什麼？`

### 預覽項目

- `活動場次`
- `費用資訊`
- `教練資訊`
- `可報名名額`

### 預覽小字
`LINE Login 後解鎖`

### 主按鈕
`查看活動場次 / 費用資訊`

### 區塊底部補充
`想先確認時間、地點與費用，再決定要不要進場，這就是你該點的地方。`

# Fight Night 首頁入口與 LINE 解鎖改版說明

## 改版目的
首頁不再直接顯示價格方案。

新的做法是：

1. 首頁只提供 `查看活動場次 / 費用資訊` 入口
2. 使用者進入 `/offers` 頁後，先看到 `LINE 會員專屬內容`
3. 透過 LINE Login 解鎖後，才能查看完整課程體系、活動場次、費用資訊、教練資訊與名額狀態

這樣做的目的不是故弄玄虛，而是讓首頁先保持品牌節奏，不過早掉進價格比較，同時先讓使用者知道這不是一次活動，而是一套可持續投入的課程系統。

這套系統現在還要再明確補上一層價值：

- 教練來自海外，是職業綜合格鬥選手
- 每一次課程都是不同的體驗以及學習
- 內容從拳擊、踢拳到防身術，形成完整系統教學

## 改版核心

### 1. 首頁改成單一入口
首頁不再出現：

- 價格
- 方案卡
- Pass 比較

首頁改成顯示：

- 活動資訊入口標題
- `LINE 會員專屬內容` 提示
- 解鎖內容預覽
- CTA：`查看活動場次 / 費用資訊`

而這個入口實際上要傳達的不只是「去看價格」，而是：

- 這是不是一套有系統的課
- 這四堂課到底在學什麼
- 這跟一次性活動有什麼不同
- 教練到底有什麼背景
- 為什麼每一次上課都值得持續回來

### 2. /offers 頁改成會員專屬內容頁
`/offers` 不再直接表現成公開報價頁，而是先告訴使用者：

- 這裡是 LINE 會員專屬內容
- 登入後會解鎖哪些資訊
- 下方各區目前是鎖定狀態

更重要的是，`/offers` 必須重新定位成：

- 課程體系介紹頁
- 活動場次／費用資訊頁
- 解鎖後的報名入口頁

其中「課程體系介紹頁」不能只是抽象地講訓練，而是要具體講出：

- 海外職業綜合格鬥選手帶課
- 每次課程內容都不同
- 拳擊、踢拳、防身術是一套連貫系統

### 3. 解鎖語氣要一致
所有相關 CTA 與區塊文案都要統一成：

- `查看活動場次 / 費用資訊`
- `快速登入查看`
- `LINE Login 後解鎖`
- `LINE 會員專屬內容`

不要再混用：

- `立即搶位`
- `選擇你的 Pass`
- `方案與報價`
- `回到方案與報價`

## 使用者流程

1. 在首頁看到 `查看活動場次 / 費用資訊`
2. 點進 `/offers`
3. 在頁首看到 `LINE 會員專屬內容`
4. 先看會解鎖哪些內容
5. 點 `快速登入查看`
6. 完成 LINE Login 後解鎖完整內容

這裡的「完整內容」不能只理解成價格，而要包含：

- 四堂課完整訓練體系
- 活動場次
- 費用資訊
- 教練資訊
- 可報名名額

而在「四堂課完整訓練體系」裡，應再清楚讓使用者理解：

- 不是四堂重複課
- 是四堂不同主題與不同學習重點
- 最終會形成拳擊、踢拳到防身術的完整基礎

## 首頁改版內容

### 原本
首頁 `TicketSection` 原本仍帶有：

- 價格遮罩
- Pass 卡片
- 方案導向語氣

### 現在
首頁 `TicketSection` 改成：

- 單一入口區塊
- 四個解鎖內容預覽
- 單一主按鈕
- 不再出現任何價格或方案名稱

### 首頁文案方向
首頁這段要讓人感覺：

- 我還沒看到價格
- 但我知道點進去會看到什麼
- 這是一個需要登入後查看的資訊入口
- 而且這不是一次活動，而是一套完整訓練體系
- 教練還具備足夠強的專業背景，值得我往下了解

## /offers 頁改版內容

### Hero 區
Hero 不再直接說「這裡有完整方案、報價」。

Hero 現在要先建立：

- 會員專屬感
- 解鎖感
- 快速登入的行動誘因
- 課程體系感
- 這不是一次體驗的認知
- 教練專業感

### 預覽區
Hero 下方先補一段：

- 四堂課完整訓練體系
- 活動場次
- 費用資訊
- 教練資訊

讓使用者知道登入之後能得到什麼，不會只有一個空泛的登入要求。

### 新增課程體系區
`/offers` 應新增一個比費用資訊更前面的區塊，專門處理：

- 這不是一次活動體驗
- 這是一套四堂課的完整訓練體系
- 教練來自海外，是職業綜合格鬥選手
- 每次課程都是不同的體驗以及學習
- 能從拳擊、踢拳一路學到防身術的完整基礎

這個區塊的作用是先建立價值，再談費用。

### 課程體系區建議內容

- 主標：`這不是一次活動，是一套完整訓練體系`
- 副標：`四堂課，從拳擊、踢拳到防身術，帶你進入完整的一套系統教學。`
- 四堂課架構卡片
- 每堂課的學習重點
- 教練背景區塊或標籤
- 鎖定浮層：`登入後查看完整四堂課內容`

### 下方正式內容區
下方各區改成：

- 區塊本體保留
- 內容做 blur / lock 狀態
- 上面蓋一層解鎖浮層
- 每區都有 `快速登入查看`

這樣可以明確表達：

- 內容是存在的
- 但現在還沒解鎖
- 下一步就是 LINE Login

## 文案原則

### 首頁
首頁語氣要像入口，不像購買。

適合：

- `查看活動場次 / 費用資訊`
- `LINE 會員專屬內容`
- `LINE Login 後解鎖`

不適合：

- `立即搶位`
- `立即購買`
- `選擇你的 Pass`

### /offers 頁
`/offers` 語氣要像專屬內容頁，不像公開價目表。

適合：

- `快速登入查看`
- `登入後會解鎖這些內容`
- `四堂課完整訓練體系`
- `海外職業綜合格鬥選手帶課`
- `每一次課程都是不同的體驗與學習`
- `登入後查看完整費用資訊`
- `登入後查看完整活動場次`

不適合：

- `方案與報價`
- `往下看完整方案`
- `回到方案與報價`

目前這次改的是：

- 文案
- 資訊架構
- 視覺上的 locked / unlocked 流程

尚未真正完成的是：

- 真實 LINE Login 驗證

也就是說，現在頁面已經會正確表達「需要 LINE Login 才能看」，但還沒有真的接上 LINE 身分驗證。

1. 決定 LINE Login 的真實串接方式

4. 解鎖後再決定是否保留 blur overlay，或直接切成完整可互動內容

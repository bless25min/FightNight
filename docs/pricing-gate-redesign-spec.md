# Fight Night 首頁方案隱藏與 LINE 查看改版說明

## 改版目的
目前首頁直接公開方案與報價，使用者雖然能快速看到價格，但也太早進入比較心態，會削弱整體品牌敘事與進場感。

這次改版的目標是：

1. 保留首頁轉換力
2. 把完整方案與報價收進 LINE 登入後頁面
3. 讓價格、教練、場次、名額形成同一條轉換路徑
4. 讓使用者覺得自己是在「進場」，不是在看一張價目表

## 改版核心策略

### 1. 首頁方案區改成 teaser
首頁仍保留三張 Pass 卡片，但不再完整公開：

- 價格做模糊或遮罩
- feature 只露出前段，剩餘內容模糊處理
- 所有卡片 CTA 統一改成 `LINE 登入查看方案與報價`

這樣的作用是：

- 仍然讓使用者知道有三種進場方式
- 不會過早把注意力縮成價格比較
- 讓使用者產生「登入後才看得到完整資訊」的心理預期

### 2. 完整資訊移到登入後頁面
登入後頁面承接首頁興趣，集中呈現：

- 完整方案與報價
- 教練資訊
- 本月仍有名額的場次
- 三館對應入口

這頁不只是一張價格頁，而是一張「進場資訊頁」。

## 使用者流程

1. 使用者在首頁看到方案 teaser
2. 點擊 `LINE 登入查看方案與報價`
3. 完成 LINE 登入
4. 進入方案與報價頁
5. 看到完整價格、教練資訊與可報名場次
6. 點擊對應場館 LINE 預留名額

## 首頁改版內容

### 現況
目前首頁 `TicketSection` 直接顯示：

- 方案名稱
- 副標
- 完整價格
- 完整 features
- 直接購票 CTA

### 改版後
首頁 `TicketSection` 改成：

- 方案名稱保留
- 副標保留
- 定位文案保留
- 價格改成遮罩顯示
- features 改成局部露出
- CTA 統一為登入查看

### 視覺建議

- 價格使用模糊層或 `NT$ ••••`
- 卡片下半部可加霧面遮罩
- 按鈕風格一致，減少「直接下單」感
- 可在卡片上加小字：`登入後查看完整內容`

## LINE 登入後頁面資訊架構

### 區塊順序

1. Hero
2. 方案與報價
3. 教練資訊
4. 本月仍有名額的場次
5. 場館資訊
6. 頁尾 CTA

### 為什麼這樣排

- Hero 先建立「你已經更靠近進場」的感受
- 方案與報價先回答最直接問題
- 教練資訊補足專業感與信任感
- 場次與名額承接行動壓力
- 場館資訊對應實際報名入口

## 場次呈現邏輯

### 固定規則

- 每月第一個星期五
- 晚上 10:00
- 1 個場次
- 3 個場館同步開放

### 呈現建議
不要把場次寫成一般課表，而要強調它的稀缺性：

- `每月只開一晚`
- `第一個星期五，晚上 10:00`
- `三館同步開放`
- `額滿就等下個月`

### 名額狀態建議

- `仍可報名`
- `名額緊張`
- `即將額滿`
- `本月已額滿`

## 教練資訊呈現邏輯
教練資訊不應該只是履歷羅列，而是要服務這個品牌敘事：

- 教練不是單純帶完一堂課
- 教練是把人安全地帶進狀態
- 課程不是即興，而是被編排過的

因此文案與畫面都應該強調：

- 專業
- 控制力
- 帶動力
- 安全邊界

## 文案語氣原則

### 應該要有的感覺

- 有門檻感
- 但不高冷
- 有被邀請進內圈的感覺
- 不是電商口吻
- 不是健身房價目表口吻

### 應避免的方向

- `立即購買`
- `優惠方案`
- `限時特價`
- `立即下單`

### 建議 CTA 語氣

- `LINE 登入查看方案與報價`
- `查看本月仍有名額的場次`
- `用 LINE 預留名額`
- `選你要怎麼進場`

## 實作建議

### 首頁
建議沿用現有 `TicketSection`，但做以下調整：

- 新增未登入 teaser 狀態
- 方案價格改成遮罩文案
- 按鈕 href 改為 LINE 登入入口
- features 顯示縮減

### 新頁面
新增登入後頁面，例如：

- `/offers`
- 或 `/member-access`

這頁建議用獨立資料結構管理，避免和首頁 teaser 共用同一份文案造成互相牽制。

### 建議資料欄位

- `plan.name`
- `plan.subtitle`
- `plan.teaserCopy`
- `plan.fullDescription`
- `plan.price`
- `plan.ctaLabel`
- `coach.name`
- `coach.bio`
- `sessions[].venueId`
- `sessions[].date`
- `sessions[].time`
- `sessions[].capacityStatus`
- `sessions[].lineUrl`

## 驗收重點

### 首頁

- 不直接露出完整價格
- 使用者仍能理解有三種 Pass
- CTA 統一導向 LINE 登入查看

### 登入後頁面

- 能完整看到方案與報價
- 能理解教練專業與課程設計
- 能快速找到本月仍有名額的場次
- 能直接進入各館 LINE 預留
